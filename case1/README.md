目录结构的构建

PS.以下所有目录的路径都是以项目目录为基底

0. 根目录  项目目录
0.1 主页 ./index.php
0.2 数据库的配置 ./config.php
0.3 初始化的配置文件 ./init.php
0.4 404.php

============================
1. 公共目录 [public文件夹] 
    用于引入CSS/JS/FONT等文件
1.0 bootstrap 文件 ./
1.1 前台主页的样式文件 ./public/home.css 
1.2 后台主页的样式文件 ./public/admin.css 
1.3 ....

============================
2. 功能组件  [com文件]
用于存放网站的各种功能模块

============================
3.图片 [imgs 文件]
用于存放网站自己的背景图/logo/按钮图/图标

============================
4.文件上传  [uploads文件夹]
用于保存上传上来的图片

============================
5.记录 [note文件夹]
用于存放 网站的说明文件/SQL文件/数据库备份

============================
6.后台部分 [admin文件夹]
用于实现网站的后台
